Vol/Range Forecast Backtester — v2 · adaptive selector

Built on the shared horizon-agnostic core (forecastCore.js). One entry primitive, one fill walker, one day-type score — reused across daily / weekly / 20-day. The adaptive selector decides per window whether to fade the median, fade the 75p, or follow a break, from a day-type score T. It runs head-to-head against each fixed leg on the same in-sample / out-of-sample split — the test is whether adaptive beats the best fixed leg on OOS Sharpe with enough trades.
low T → range day
above → follow
adaptive = the day-type selector (the v2 brain). fade75 / fadeMed = always fade that band. follow = always break-and-go. The row that matters is adaptive vs the best fixed leg on OOS Sharpe at a non-trivial OOS trade count. If adaptive only wins in-sample, or wins OOS on <30 trades, the selector is curve-fit — not edge. Costs and a true later-period OOS split are applied; numbers are intentionally harsh.
