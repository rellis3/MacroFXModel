Confluence Mode Global — all instruments · dashboard · backtest · alerts
Price selectionmode
Where to draw a confluence line when two Fibs overlap. Midpoint = average of the two prices. Lowest/Highest matches the Pine Script indicator.
Cluster mergemode
On: nearby Fib pairs within the merge radius are averaged into a single level. Off: every qualifying pair gets its own line, matching the Pine Script indicator exactly.
To match the Pine Script indicator exactly: set Price selection → Lowest and Cluster merge → Off, then hit Save to KV. All consumers — dashboard, backtest, and Telegram alerts — update immediately.

Default (Midpoint + On) keeps the current dashboard behaviour: nearby Fib pairs are averaged into one level, which gives fewer, smoother lines. Switch to Pine Script mode when you want the dashboard grey lines to sit at the same price as the orange indicator lines on your chart.

5m Kalman Filter Bias T8 · Intraday
Adds a short-term mean-reversion tilt to the macro score using a Kalman filter on 5m closes. The filter tracks a time-weighted fair-value estimate; deviation from that estimate (in σ units) triggers the score. Lower Q → smoother tracking. Higher R → filter is trusted more than raw price. When normDev < −threshold the long score is added. When normDev > +threshold the short score is subtracted.
Lookback bars5m bars
Closed 5m bars fed into the filter. 40 bars ≈ 3h20m. More bars = longer memory, slower to adapt.
Process noise Qfraction
Q as fraction of price variance. Lower = filter evolves slowly (smooth). Higher = filter tracks price more aggressively.
Observation noise Rfraction
R as fraction of price variance. Higher = trust the filtered state more than each observation (smoother).
Threshold (σ)deviation
Normalised deviation that triggers a score. 1.5σ = more selective than z-score (Kalman is smoother so use higher value).
Long score weightscore pts
Score added to the macro total when normDev < −threshold (price below Kalman → long bias).
Short score weightscore pts
Score subtracted from the macro total when normDev > +threshold (price above Kalman → short bias).
