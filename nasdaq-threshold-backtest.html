System Logic: Nasdaq price is never a SIGNAL input. Gate 1A (Slow Macro Regime) reads Fed/ECB/BOJ balance sheets, RRP, TGA and HY credit spreads with zero Nasdaq input → BULLISH/BEARISH/NEUTRAL (Gate 1B, a fast intraday flow threshold, is planned but not yet built — see cogConfig.js). Gate 2 (Risk/Vol) reads QQQ/SPY price only to measure volatility/correlation regime for stop distance & sizing — never direction → VALID/INVALID + eligible risk tier. Gate 3 (Direction) reads DXY/USDJPY/ES/RTY cross-asset macro, never NQ/QQQ → LONG/SHORT/NEUTRAL. Gate 4 (Execution) is the only gate that touches NQ/QQQ price, and only to place a trade Gates 1–3 already justified. The Exit Engine re-scores every bar against the same 3 gates aligned to the trade's direction → stay in / REDUCE 50% / CLOSE. A hard stop (ATR-fraction, GARCH-sigma, or expected-move) always overrides.
