Vol-Forecast Benchmark · σ estimator OOS scorecard

The strategy stack currently assumes which volatility estimator is best per asset class (commodity→HV20, index→GARCH, fx→Yang-Zhang). This measures it. Every incumbent estimator is imported from volBacktestEngine.js (no copies) and scored on how well it predicts each instrument's realised range, against one new entrant — HAR-RV, fit walk-forward. Scored the same honest way strategies are: a true in-sample / out-of-sample split, ranked by out-of-sample QLIKE (lower = better). σ is the ruler everything downstream — bands, "extension past the mean" — is measured in, so a better σ sharpens every idea built on it.

QLIKE = Patton's robust forecast loss r/p − ln(r/p) − 1; MSE = mean squared error on variance. Both are lower-is-better and minimised (0) by a perfect forecast. The column that matters is OOS QLIKE — in-sample fit is not evidence. A row highlighted green is that instrument's OOS winner. HAR-RV is the only new entrant; if it doesn't beat the incumbent for an asset class out-of-sample, the incumbent stays. Note: this scores the variance forecast, not a strategy — it tells you which ruler to build on, not which trade to take.
