How it works — Each month-end the engine scores 5 macro factors (Net Liquidity momentum, Yield Curve, HY Credit Spread, 10Y TIPS Real Yield, ISM PMI) using rolling 252-day z-scores. The composite score sets a base allocation: >High → 100%, >Mid → 75%, >Low → 50%, else 25%. A trend filter (200-day MA + 12-month momentum) scales it: both bullish = 1.0×, either = 0.8×, neither = 0.55×. A VIX sizer applies a final factor: LOW = 1.0×, NORMAL = 0.85×, HIGH = 0.60×, EXTREME = 0.30× (no hard block). Floor is configurable (default 50%). TLT hedge uses an inverted signal — weak macro → high bond allocation (floor 15%). Toggle Russell 2000 and TLT in the run bar. Data: OANDA D1 (QQQ/SPY/IWM/TLT proxy), FRED REST API for macro series, Yahoo Finance for VIX. Publication lags prevent lookahead bias.
