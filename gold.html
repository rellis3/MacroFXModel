Overview

The Gold Bot is a mean-reversion + macro-confluence system for XAU/USD. It combines a two-layer macro model (fundamental direction) with a Fibonacci zone system (where to enter) and a VuManchu oscillator assessment (when momentum confirms). A trade is only valid when all eight gates pass simultaneously.

Key Price Levels

Daily Pivot (PP) — (prevH + prevL + prevC) / 3. Primary mean-reversion anchor. Static all day.
R1 / S1 — First resistance / support extensions from daily pivot. Directional targets and entry zones.
4H Pivot — Same formula, 4-hour bars. Sub-daily precision confluences. Displayed in Zone Ticker.
Daily Open — Psychological anchor. Price frequently reverts to it intraday. Tagged when price approaches.
VAH / VAL — Value Area High / Low from previous day's volume profile. Key consolidation zone boundaries.
Session VWAP — Cumulative today's VWAP. Above = buyers in control; below = sellers. Chop band gate.
Asia Body-Range Fibs — 00:00–06:00 London. Fib levels projected from session body. Today + yesterday confluences.
Monday Body-Range Fibs — Weekly range anchor. Current + previous Monday body. Cross-session matches are high-star.

Sniper Suite Confluence Box

Three structural biases are tracked by the bot and displayed in the Zone Ticker. All three aligning = highest conviction.

  • Pivot Bias — price above/below daily pivot PP. BULL = price > PP; BEAR = price < PP.
  • Structural Bias — CHoCH / BOS analysis on 30m bars. Detects trend breaks and continuations.
  • Momentum — WT crossover direction. BULL = WT1 crossing above WT2; BEAR = crossing below.

Entry Sequence

  1. All 7 checklist gates must pass (or warn on non-critical gates — stability, VWAP chop).
  2. Identify the highest-star Fib confluence or daily pivot in the signal direction.
  3. Wait for price to enter the Golden Pocket (0.618–0.65) of the zone. Zone Ticker activates automatically.
  4. Zone Ticker VMU must return AGREE (≥2/3 components).
  5. Sniper Suite confluence box: Pivot Bias + Structural Bias must align with trade direction. Momentum is the trigger.
  6. Session VWAP must be on the correct side: price above VWAP for long; price below VWAP for short.
  7. VWAP chop gate must be clear (price > ATR×0.25 from VWAP).
  8. Enter on WT cross (momentum trigger) or on the candle that confirms VMU AGREE.

Non-Entry Conditions (skip the trade)

  • Any hard-fail gate (regime LOW, signal NEUTRAL, L2 momentum <2/4).
  • Price inside VWAP chop band — no directional conviction.
  • Price >60 pips from any aligned level — do not chase.
  • COT leveraged funds >80th percentile — mean-reversion risk elevated.
  • GARCH daily cap already >85% used — limited remaining range.
  • VMU returns OPPOSE (oscillators actively contradicting trade direction).
  • OTC forecast = CHOPPY session character — sessions historically close near open.
  • Sniper Suite bias box: Pivot + Structural disagree — wait for alignment before entry.

Why we don't use Pine Script's simple percentile range

The bot's GARCH(1,1) + OTC Forecast is superior: it regime-matches the historical sample to current volatility (LOW/NORMAL/HIGH), computes CI68/CI95 confidence intervals, tracks remaining range cap intraday, and flags CHOPPY vs TRENDING session character. The Pine Script uses a simple 90-day percentile of open-to-high % — useful but a significant simplification of what the OTC engine already does.
